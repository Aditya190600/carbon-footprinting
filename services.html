Analysis

Native Javascript is used to create an interactive map which lets people search for an address. Then all the places around the world with the same name would be displayed and the user would have to choose from them. Once the place is selected, that place is pointed out in the map with a marker. A label is attached to the marker which shows the latitude and longitude of that place. This was used to get latitude and longitude values of an address which would be then be used to assess the carbon monoxide levels in that place.

Carbon at your Location

This could be further integrated with a machine learning model which would predict the Carbon Monoxide levels in a specific area. The output values i.e latitude and longitude values of this model would be then sent to the machine learning model as an input.
